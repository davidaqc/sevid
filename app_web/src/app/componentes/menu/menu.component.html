Bienvenido al Sistema de Verificación de Identidades (SEVID)
Las funcionalidades del sistema son:
  1. Captura: Permite tomar una fotografía y guardarla en el dispositivo, así como visualizar una imagen en tiempo real.
    • Tomar Fotografía
    • Visualizar Imagen
  2. Genera: Permite generar datos a partir de una imagen pre-cargada en la sistema o bien, a través de una imagen que se envie desde la aplicación web.
    • Seleccionar Archivo
    • Enviar Imagen
    • Utilizar Imagen Pre-Cargada
  3. Resultado: Permite visualizar los datos extraídos de la imagen e indicar su validez.
    • Solicitar Datos
